# Hex to RGB API

This project is a simple REST API built with Express.js that converts HEX color codes to RGB.

## Project Structure

- `src/` - source code (server, routes)
- `test/` - unit and integration tests
- `.gitignore` - ignores node_modules and other unnecessary files
- `package.json` - project metadata and dependencies

## How to Run

```bash
npm install
npm start
